Q83 -- Remove Duplicates From Sorted List

Posted by daisyliu0225 on January 30, 2025

題目敘述

Given the head of a sorted linked list, delete all duplicates such that each element appears only once. Return the linked list sorted as well.

Constraints:

  1. The number of nodes in the list is in the range [0, 300].
  2. -100 <= Node.val <= 100
  3. The list is guaranteed to be sorted in ascending order.

難度標記：easy

解題思路

Step 1

這題我的想法很簡單，我設一個變數叫current val，如果今天node val跟current val一樣的話，就要把這個node刪掉

會有這樣的想法是因為題目有有說linked list已經是sorted了，所以如果current val有出現過， 那就代表前面的node一定是current val，現在這個node一定是duplicate，所以現在的node要刪掉。

如果node val跟current val不一樣的話，代表這一個node val第一次出現，同時也進到新的node val的區域裡，所以把current val改成新的node val

Step 2

至於刪掉node的方法如何處理，我的想法是有一個ptr指著前一個node(我叫它curNode)，還有一個ptr叫head，紀錄現在走到哪一個node

會這樣做是因為由題目的structure可以知道這是一個單項的linked list，所以過了前一個node就回不來了， 但刪除node時需要前一個node跟後一個node進行連接，不然會runtime error，所以就設置一個prevNode來記錄上一個node。 至於下一個node只要用head->next就可以處理了。

流程就是：把前一個node連接到下一個node(head->next)->刪除head->把head指到前一個node的下一個

Step 3

至於return的東西，我們在程式剛開始的時候直接設一個指標叫ret，指到剛傳進來的head，最後return ret就好了

Debug/遇到的問題

剛開始runtime error，結果發現我constraint沒看，node值在-100~100間，修好就過了

Whole code


ListNode* deleteDuplicates(ListNode* head) {
    int cur_num = 101;
    ListNode* prevNode = head;
    ListNode* ret = head;
    while(head != nullptr){
        if(head->val != cur_num){
            cur_num = head->val;
            prevNode = head;
            head = head->next;
        }
        else{
            prevNode->next = head->next;
            delete head;
            head = prevNode->next;
        }
    }
    return ret;
}
