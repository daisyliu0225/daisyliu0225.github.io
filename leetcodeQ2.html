Q2 -- Add Two Numbers

Posted by daisyliu0225 on January 25, 2025

題目敘述

You are given two non-empty linked lists representing two non-negative integers. The digits are stored in reverse order, and each of their nodes contains a single digit. Add the two numbers and return the sum as a linked list.
You may assume the two numbers do not contain any leading zero, except the number 0 itself.

Constraints:

  1. The number of nodes in each linked list is in the range [1, 100].
  2. 0 < = Node.val < = 9
  3. It is guaranteed that the list represents a number that does not have leading zeros.

難度標記：medium

解題思路

這一題有參考別人的答案
其實看到linked list的時候我就有想到要把每一個node相加起來，並且算carry。
但奈何方法太笨，效率太低，所以debug了幾次code之後就看一下別人如何解

Step 1

首先，先審題，了解題目要什麼是我們必須知道的。題目要求要給一個ListNode*，在上面就有給一點點提示


/*
 Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
*/
依照上面的提示我們就可以知道，ListNode裡面有一個變數叫val，還有一個變數叫next，是一個指到ListNode的pointer

還有三個函數，主要就是幫ListNode的Node進行初始化

第一個函數是創立一個新的Node，裡面的val是0，然後下一個Node是nullptr，也就是沒有接到任何node

第二個函數是創立一個新的Node，裡面的val是x，下一個Node是nullptr

第三個函數是創立一個新的Node，裡面的val是x，但下一個Node有被決定

在建立這些Node的時候其實也可以都不指定，因為之後可以用(nodename)->val / (nodename)->next 進行指定

建立Node的時候要記得這樣做：ListNode* (nodename) = new ListNode();，不然程式會不認

Step 2

接著就是進行node之間的相加了，做法自然就是把兩個linked list loop過一次就可以知道相加之後的和

首先我們先創造一個dummyNode，至少讓程式知道要從哪裡開始跑。

那問題就來了，在程式執行的時候我們並不知道哪一個linked list比較長，所以我就想說只有在兩個有下一個(->next != nullptr)的時候才做while，剩下我就分case處理

別人高明的地方就是這個！！他全部都用while處理，只是while終止條件是當我沒有要執行任何東西的時候才終止(l1 != nullptr || l2 != nullptr || carry != 0)

那如果今天某一個是nullptr但另一個不是該如何處理呢？那就是直接把nullptr的代表值設成0就好了。這個方法真的很聰明。

Step 3

後面就是相加之後的一些處理了，想法很簡單。

如果今天兩者相加大於等於10的話，那就是有carry。carry就要被設成1。

至於carry會不會超過1，其實我認為不會，原因是當兩個node都剛好是9，而且carry是1的時候，值也只有19，所以可以知道carry最大只有1。

Step 4

最後的最後，還要處理下一個是不是nullptr，那判斷方法就是如果現在的ptr是nullptr的話，那就讓它繼續是nullptr。

如果不是的話，那就是讓現在的node跳到下一個。

Step 5

來到回傳node的地方了。因為dummyNode裡面是空的，只有幫我們指到第一個node是誰，所以回傳dummyNode的下一個node就好了

Debug/遇到的問題

其實我做while判斷的跳出結果不用一直鎖l1->next != nullptr，只要讓l1 != nullptr就可以了
有遇到其他狀況我再處理

幹嘛執著在一定要把第一個node傳回去，其實只要有linked list，不管傳第幾個node，只要有正確結果就好了
換言之，我可以設一個dummyNode然後再傳回dummyNode的下一個就好

一個小筆記：
if(...) a1 = (condition 1) else a1 = (condition 2) 可以簡化成
a1 = (...) ? (condition 1) : (condition 2)

Whole code


ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
    ListNode* dummyHead = new ListNode(0);
    ListNode* curNode = dummyHead;
    int carry = 0;

    while(l1 != nullptr || l2 != nullptr || carry != 0){
        int val1 = (l1 != nullptr) ? l1->val : 0;
        int val2 = (l2 != nullptr) ? l2->val : 0;
        int sum = val1 + val2 + carry;
        int digit = sum % 10;
        carry = sum / 10;

        ListNode* newNode = new ListNode(digit);
        curNode->next = newNode;
        curNode = curNode->next;

        l1 = (l1 != nullptr) ? l1->next: nullptr;
        l2 = (l2 != nullptr) ? l2->next: nullptr;
    }
    return dummyHead->next;
    }
