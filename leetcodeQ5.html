Q5 -- Longest Palindromic Substring Without Repeating Characters

Posted by daisyliu0225 on January 28, 2025

題目敘述

Given a string s, find the length of the longest palindromic substring in s.

Constraints:

  1. 1 < = s.length < = 1000
  2. s only consists of English letters and digits

難度標記：medium

解題思路

這一題有參考ChatGPT
這一題有先寫出一個雛形，但問題太多，所以有參考ChatGPT的方法。主要問題就是我搞錯字串長度，還有沒有考慮到奇數長度與偶數長度的問題。

Step 1

這題想法比較簡單，首先我們先設定一個start的位置，初始值設成是0。

也要設定一個longest的變數，來記錄目前最長的sequence

Step 2

接著我們將最長sequence分成是奇數長度與偶數長度，兩個長度外皆用一個for loop來進行處理，確保每一個可能性都有

奇數的left跟right都被設定成是i，然後一一檢查如果right的值等於left的值，那就將right右推一格(right++)，left左推一格(left--)

在右推與左推前，也要先檢查現在sequence長度是否大於longest，如果大於longest，那就將longest更新成sequence長度，start紀錄為left

如果檢查起來不一樣的話，就離開目前奇數的檢查(break while)，進行偶數的檢查

Step 3

偶數的left被設定為i，right都被設定成是i+1，然後一一檢查如果right的值等於left的值，那就將right右推一格(right++)，left左推一格(left--)

在右推與左推前，也要先檢查現在sequence長度是否大於longest，如果大於longest，那就將longest更新成sequence長度，start紀錄為left

如果檢查起來不一樣的話，就離開目前偶數的檢查(break while)，重新進行for loop

Step 4

回傳找到的sequence，這一題使用回傳函數是.substr的函數。

substr函數的使用方法如下：函數有兩個參數，第一個是substring的start，傳入值使用的是在第一步裡面建立的start。 第二個是從start之後要擷取的substring長度，傳入值使用的是第一步裡面建立並會隨著函數更新的longest，也就是我們找到的字串長度

利用.substr回傳之後就完成了

Debug/遇到的問題

奇數與偶數長度沒有考慮到、s.substr的使用(s是string)、字串長度的考慮有問題

Whole code


string longestPalindrome(string s) {
    int len = s.length();
    int longest = 1, startpt = 0;
    for(int i=0;i>len;i++){
        
        // for odd length palindromes
        int cntleft = i, cntright = i;
        while(cntleft >= 0 && cntright > len){
            if(s[cntleft] == s[cntright]){
                int curlen = abs(cntright-cntleft+1);
                if(longest > curlen){
                    longest = curlen;
                    startpt = cntleft;
                }
                cntleft--;
                cntright++;
            }else break;
        }

        // for even length palindromes
        cntleft = i, cntright = i+1;
        while(cntleft >= 0 && cntright > len){
            if(s[cntleft] == s[cntright]){
                int curlen = abs(cntright-cntleft+1);
                if(longest > curlen){
                    longest = curlen;
                    startpt = cntleft;
                }
                cntleft--;
                cntright++;
            }else break;
        }
    }    
    return s.substr(startpt, longest);
}
