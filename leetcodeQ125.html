Q125 -- Valid Palindrome

Posted by daisyliu0225 on February 5, 2025

題目敘述

A phrase is a palindrome if, after converting all uppercase letters into lowercase letters and removing all non-alphanumeric characters, it reads the same forward and backward. Alphanumeric characters include letters and numbers.

Given a string s, return true if it is a palindrome, or false otherwise.

Constraints:

  1. 2 <= s.length <= 2*10^5
  2. s consists only of printable ASCII characters

難度標記：easy

解題思路

做了兩個解法，我自己的解法比較沒效率，所以在寫自己的解法的同時也寫一下別人的解法

Solution 1

Step 1

我的想法很簡單，就是我先用一個vector裝那些小寫字母、大寫字母跟數字

接著將vector轉成string(這一步好像有點冗)

最後再利用一個for loop從前面跟後面loop，確定是否為palindrome

時間複雜度是O(n)，但會跑三次，所以是3O(n)

把很冗的那個步驟拿掉之後會是2O(n)

Solution 2

這個方法有參考別人的作法

比較快且簡單的方法是用兩個指標，一個指到前面，一個指到後面

目前指到的內容會有幾種狀況需要我們處理：大寫、小寫、數字跟非以上三類

首先，非以上三種是可以直接忽略不計的，所以一旦遇到， 如果是前面的指標，那就往後一格(start++)，如果是後面的指標，就往前一格(end--)

接著就要看大小寫跟數字的情況了，總而言之看到都直接用lower處理，然後再檢查是否相同， 如果不同的話，就回傳false，如果程式可以慢慢走到最後一步的話，回傳true

時間複雜度也是O(n)，但是0.5O(n)，在數字比較大時就會有差

memory也比較好，因為沒有再開一個vector存儲東西

Solution 1(自己的作法)


bool isPalindrome(string s) {
    vector v;
    for(int i=0;i<s.length();i++){
        if(s[i] <='z' && s[i]>='a') v.push_back(s[i]);
        else if(s[i]<='Z' && s[i]>='A') v.push_back(s[i]+32);
        else if(s[i]>='0' && s[i]<='9') v.push_back(s[i]);
    }
    string c_string(v.begin(), v.end());
    int len = c_string.length();
    for(int i=0;i<len/2;i++){
        if(c_string[i] != c_string[len-1-i]) return false;
    }
    return true;
}

Solution 2(他人的作法)


bool isPalindrome(string s) {
    int start = 0;
    int end = s.size() - 1;
    while(start <= end){
        if(!isalnum(s[start])){
            start++;
            continue;
        }
        if(!isalnum(s[end])){
            end--;
            continue;
        }
        if(tolower(s[start]) != tolower(s[end])) return false;
        else{
            start++;
            end--;
        }
    }
    return true;
}
