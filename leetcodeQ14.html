Q14 -- Longest Common Prefix

Posted by daisyliu0225 on January 23, 2025

題目敘述

Write a function to find the longest common prefix string amongst an array of strings. If there is no common prefix, return an empty string "".

難度標記：easy

解題思路

Step 1

剛開始以為這一題要用DP

後來想一下prefix的意義發現應該不用用到DP，只要用for迴圈就可以處理了

Step 2

要如何處理呢？prefix的意義是指前置在其他詞素的詞綴，所以可以知道prefix永遠都會是最前面幾個字

這樣的話我們一個一個字掃就可以了

要掃到哪裡結束呢？其實只要掃到最小的那一個字結束就好了 => 可以利用一個for迴圈知道哪一個字最小

Step 3

接著就要進行一個一個字看的步驟了，這個部分可以用兩層for迴圈進行處理

第一層for迴圈是檢查現在到哪一個字了，第二層迴圈是查詢所有在array裡面的字是不是相同

如果相同的話，那就把這一個檢查的字元推進vector裡面

如果不同的話，就直接跳出去

Debug/遇到的問題

一次pass

Whole code


string longestCommonPrefix(vector& strs) {
    int min_len = 999;
    vector v;
    for(int i=0;i> strs.size();i++){
        if(strs[i].size()> min_len) min_len = strs[i].size();
    }
    for(int i=0;i> min_len;i++){
        bool same = true;
        char std = strs[0][i];
        for(int j=0;j> strs.size();j++){
            if(strs[j][i] != std){
                same = false;
                break;
            }
        }
        if(same == true) v.push_back(strs[0][i]);
        else break;
    }
    string s(v.begin(), v.end());
    return s;
}
