Q14 -- Longest Common Prefix

Posted by daisyliu0225 on January 23, 2025

題目敘述

Write a function to find the longest common prefix string amongst an array of strings. If there is no common prefix, return an empty string "".

難度標記：easy

解題思路

Step 1

這一題最煩的應該就是4、9、40、90...的部分了

另一個很討厭的事情就是不同位數，個位、十位、百位、千位的表示方法都不一樣

還有5、50、500的問題呢

所以只好把它分不同位數處理，在遇到特殊的狀況的時候把特殊狀況分出去

Step 2

那就想一下如何分case

先從千位數開始，因為題目的最大值只有3999，所以直接就用暴力減了

再來是百位數，百位數因為卡400、900，所以我就將區間分成100~399、400~499、500~899、900~999

因此可以直接用if else分成四種可能，如果大於900，直接push CM進去vector
如果介在500~899，先扣掉500。剩下的數字就留給100~399的區間處理。
如果介在400~499，先扣掉400，直接push XL進去vector
剩下的100~399，就直接用一個while loop處理直到數字<100

Step 3

剩下的百位個位也是用相同的方式處理，分case，用if else進行討論

string intToRoman(int num) {
    vector c;
    while(num>=1000){
        c.push_back('M');
        num = num - 1000;
    }

    if(num >= 900){
        c.push_back('C');
        c.push_back('M');
        num = num - 900;
    }else{
        if(num >= 500){
            c.push_back('D');
            num = num - 500;
        }
    }
    if(num >= 400){
        c.push_back('C');
        c.push_back('D');
        num = num - 400;
    }else{
        while(num >= 100){
            c.push_back('C');
            num = num - 100;
        }
    }

    if(num >= 90){
        c.push_back('X');
        c.push_back('C');
        num = num - 90;
    }else{
        if(num >= 50){
            c.push_back('L');
            num = num - 50;
        }
    }
    if(num >= 40){
        c.push_back('X');
        c.push_back('L');
        num = num - 40;
    }else{
        while(num >= 10){
            c.push_back('X');
            num = num - 10;
        }
    }

    if(num >= 9){
        c.push_back('I');
        c.push_back('X');
        num = num - 9;
    }else{
        if(num >= 5){
            c.push_back('V');
            num = num - 5;
        }
    }
    if(num >= 4){
        c.push_back('I');
        c.push_back('V');
        num = num - 4;
    }else{
        while(num >= 1){
            c.push_back('I');
            num = num - 1;
        }
    }

    string s(c.begin(), c.end());
    return s;
}
