Q1 -- Two Sum

Posted by daisyliu0225 on February 4, 2025

題目敘述

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

You can return the answer in any order.

Constraints:

  1. 2 <= nums.length <= 10^4
  2. -10^9 <= nums[i] <= 10^9
  3. -10^9 <= target <= 10^9
  4. only one valid answer exists

難度標記：easy

解題思路

這一題原本用暴力解，時間複雜度會是n^2，後來看了別人的解答用hash table，以下紀錄hash table的作法

Step 1

Hash Table是什麼已經寫在下面了。

這一題使用hash table的想法是利用hash table的特性在hash table中快速查找是否有對應的sum

Step 2

首先先建立一個hash table並把vector中的東西loop一遍， 在loop的時候，我們同時尋找當前的element的complement，也就是target-element的數字有沒有在map裡面

會有兩種狀況，沒有或有：如果沒有的話，那就先把hash table裡面的那一個element的數字設成i， 如果有的話，那就是將那一個element的complement代表的數字跟現在element一同回傳

因為題目已經確認只有一個解了，但如果可以有多解的話。回傳的部分可以先用一個vector存著， 等全部loop過再回傳。

Step 3

如果都沒有找到的話，回傳空集合

Debug/遇到的問題/觀念紀錄

Hash table是啥

Hash Table是一個data structure，它提供資料快速的查找、增添以及刪除。 可以達成如此高的效率是因為它使用的是key-value pair的技術。

Hash table的特性

  1. Key value：每一個value都有特殊的key對應
  2. Hash Function：每一個數學方程都會將一個key對應到一個index， 這個function必須將key平均分配不然會有collision
  3. Fast Lookup(平均是O(1)complexity)： 因為hash function的關係，找到每一個key只需要O(1)的複雜度

一個hash table的例子：
程式設計師可以自己設計一個hash function，像這樣： 


int hashFunction(string key, int tableSize) {
    int hash = 0;
    for (char ch : key) {
        hash += ch;  // Simple summing of ASCII values
    }
    return hash % tableSize;  // Ensuring the index stays within bounds
}
也可以交給C++裡面的unordered map進行代管，由C++內部系統進行設計 

unordered_map ageMap;
ageMap["Alice"] = 25;
ageMap["Bob"] = 30;
ageMap["Charlie"] = 22;

cout << ageMap["Bob"];  // Output: 30

✅使用時機

  1. 當你需要快速查找東西的時候(O(1)complexity)
  2. 當你需要快速加入或刪除東西的時候
  3. 當你確定每一個value都有其對應的unique key的時候

❌不建議使用時機
如果data必須要sort之後才可以進行處理

Whole code


vector twoSum(vector& nums, int target) {
    unordered_map numMap;
    int n = nums.size();

    for(int i=0;i<n;i++){
        int complement = target - nums[i];
        if(numMap.count(complement)){
            return {numMap[complement], i};
        }
        numMap[nums[i]] = i;
    }
    return {};
}
