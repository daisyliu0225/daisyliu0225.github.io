Q70 -- Climbing Stairs

Posted by daisyliu0225 on February 5, 2025

題目敘述

You are climbing a staircase. It takes n steps to reach the top.

Each time you can either climb 1 or 2 steps. In how many distinct ways can you climb to the top?

Constraints:

  1. 1 <= n <= 45

難度標記：easy

解題思路

典型DP題

Step 1

題目中有明確的說一次只能上一格或兩格，現在用暴力解想一下會發現太複雜，所以就往DP的方向想了

那從DP方向下手的話就要尋找現在的case跟之前的case有甚麼關係

Step 2

把可能性列出來之後就會發現爬樓梯的關係就是前一個+前兩個的相加

照著這樣做就完成了

Debug/遇到的問題/觀念紀錄

runtime error有可能是long long的問題

Whole code


int climbStairs(int n) {
    int totalclimb[50] = {0};
    for(int i=1;i<=n;i++){
        if(i == 1) totalclimb[1] = 1;
        else if(i == 2) totalclimb[2] = 2;
        else totalclimb[i] = totalclimb[i-1] + totalclimb[i-2];
    }
    return totalclimb[n];
}
