Q100 -- Same Tree

Posted by daisyliu0225 on February 6, 2025

題目敘述

Given the roots of two binary trees p and q, write a function to check if they are the same or not.

Two binary trees are considered the same if they are structurally identical, and the nodes have the same value.

難度標記：easy

解題思路

Step 1

這一題如果稍微想一下真的挺簡單的，因為翻轉的樹是不可以視為同一棵樹的， 只有那些位置跟值一樣的才可以視為是相同

所以我們先檢查位置是否為一樣：左為null、右非null；左非null、右為null；左為null、右為null跟左右都非null

前面兩個狀況都可以直接回傳false，第三個狀況則是回傳true，而第四個狀況必須要再檢查左右val是否相同

下一個就是要跑左邊跟右邊，用return a & b的方法就可以了。使用"and"是因為左邊跟右邊必須相同才可以是true，不然就會是false

Debug/遇到的問題

原先的作法是檢查left是否為nullptr跟右邊是否為nullptr之後才會下一層， 但我沒有想到其實如果是左邊nullptr跟右邊nullptr也算是一種可能性

Whole code


bool isSameTree(TreeNode* p, TreeNode* q) {
    if(p == nullptr && q != nullptr) return false;
    if(p != nullptr && q == nullptr) return false;
    if(p == nullptr && q == nullptr) return true;
    if(p->val != q->val) return false;

    return isSameTree(p->left, q->left) && isSameTree(p->right, q->right);
}
