Q191 -- Number of 1 bits

Posted by daisyliu0225 on February 7, 2025

題目敘述

Given a positive integer n, write a function that returns the number of set bits in its binary representation (also known as the Hamming weight).

難度標記：easy

解題思路

Step 1

這一題的想法很簡單，就是把decimal數字變成binary。

但這一題又比一般decimal to binary的題目簡單，原因是因為它算的是1的個數，所以不用在轉換完之後再考慮哪一邊先

所以我就用一個while loop，條件設定為n>=1時不斷做這些事

首先先檢查n%2的答案是多少，如果是1的話，那1的counter就加一，如果不是1，就不理它，接著將n/2，然後又回到loop

最後回傳1的counter就完成了

Debug/遇到的問題

無

Whole code


int hammingWeight(int n) {
    int one_cnt = 0;
    while(n>=1){
        if(n%2 == 1) one_cnt++;
        n = n/2;
    }
    return one_cnt;
}
