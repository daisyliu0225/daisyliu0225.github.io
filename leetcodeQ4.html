Q4 -- Median of Two Sorted Arrays

Posted by daisyliu0225 on January 22, 2025

題目敘述

Given two sorted arrays nums1 and nums2 of size m and n respectively, return the median of the two sorted arrays.

The overall run time complexity should be O(log (m+n)).

難度標記：hard

解題思路

Step 1

先看一下我們需要處理的array有兩個特性

  1. 均為sorted(已經幫忙排序好了)
  2. 第一個array的size是m， 第二個array的size是n

Step 2

回顧一下所要求的median，median是中位數。

計算有限個數的數據的中位數的方法是：把所有的同類數據按照大小的順序排列。
如果數據的個數是奇數，則中間那個數據就是這群數據的中位數； 如果數據的個數是偶數，則中間那2個數據的算術平均值就是這群數據的中位數。

計算中位數還有一個隱藏的需求，也就是所有的數據必須要先排序好

Step 3

那就來看一下如何處理這兩個array的排序問題

題目有說到time complexity是O(log (m+n))，所以可以知道最多只能用到一次的sorting。

而且這個sorting方法只能是Merge sort、Heap Sort、Quick Sort的其中一種(均為O(logn))

通常在看到有兩個sorted array必須要併起來的問題的時候，請立刻要想到 Merge Sort

Step 4

那如何把兩個array merge起來？

其實很簡單，只要比哪一個比較小，直接塞進去就好了

Debug/遇到的問題

寫完之後很不幸的掛掉了

原因：在merge的時候沒有檢查array是否已經到底了

Whole code


double findMedianSortedArrays(vector<int>& nums1, vector<int>& nums2) {
    double median;
    int median_num = 0;
    bool even_num = 0;
    int total = nums1.size()+nums2.size();
    int num1counter = 0;
    int num2counter = 0;
    vector<int> sorted_arr;
    if(total % 2 == 1){
        median_num = total/2;
    }else{
        median_num = total/2;
        even_num = 1;
    }
    for(int i=0;i<median_num+1;i++){
            if (num1counter == nums1.size()) {
            sorted_arr.push_back(nums2[num2counter++]);
        }
        // If we've used up all elements in nums2, take from nums1
        else if (num2counter == nums2.size()) {
            sorted_arr.push_back(nums1[num1counter++]);
        }
        // Otherwise compare
        else if (nums1[num1counter] <= nums2[num2counter]) {
            sorted_arr.push_back(nums1[num1counter++]);
        } else {
            sorted_arr.push_back(nums2[num2counter++]);
        }
    }
    int sorted = sorted_arr.size();
    if(even_num == 1){
        median = (sorted_arr[sorted-1] + sorted_arr[sorted-2])/2.0;
    }else median = sorted_arr[sorted-1];
    return median;
}
