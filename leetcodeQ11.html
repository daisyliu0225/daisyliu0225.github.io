Q11 -- Container With Most Water

Posted by daisyliu0225 on January 30, 2025

題目敘述

You are given an integer array height of length n. There are n vertical lines drawn such that the two endpoints of the ith line are (i, 0) and (i, height[i]).

Find two lines that together with the x-axis form a container, such that the container contains the most water.

Return the maximum amount of water a container can store.

You should not slant the container.

Constraints:

  1. n == height.length
  2. 2 <= n < 10^5;
  3. 0 <= height[i] < 10^4;

難度標記：medium

解題思路

有看一下別人的解答。原因：我沒有想到要如何尋找兩邊的條件，讓最終的值為最大值

Step 1

這一題的解答比我想像中簡單

提示裡面已經寫可以用兩個指標作為左、右標記來尋找最大值，那要如何利用兩個標記尋找

其實就是比左右標記的大小，今天如果右邊比較大，就移動左邊，如果左邊比較小，就移動右邊

移動完標記之後算出矩形的面積，面積以左右標記的距離*min(左標記, 右標記)計算，如果今天結果比目前計算的最大面積大的話，最大面積就進行更新

最後回傳最大面積即可

Debug/遇到的問題

剛開始就是沒有想到尋找兩邊的條件，後面程式不小心搞錯左右移動的條件，但有成功自己克服

Whole code


int maxArea(vector<int>& height) {
    int left = 0, right = height.size()-1;
    int max_area = 0;
    while(left != right){
        int cur_area = abs(left-right)*min(height[left], height[right]);
        if(cur_area > max_area) max_area = cur_area;
        if(height[left] > height[right]) right--;
        else left++;
    }
    return max_area;
}
