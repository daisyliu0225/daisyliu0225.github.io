Q8 -- String to Integer (atoi)

Posted by daisyliu0225 on February 1, 2025

題目敘述

Implement the myAtoi(string s) function, which converts a string to a 32-bit signed integer.

The algorithm for myAtoi(string s) is as follows:

Constraints:

  1. Whitespace: Ignore any leading whitespace (" ").
  2. Signedness: Determine the sign by checking if the next character is '-' or '+', assuming positivity if neither present.
  3. Conversion: Read the integer by skipping leading zeros until a non-digit character is encountered or the end of the string is reached. If no digits were read, then the result is 0.
  4. Rounding: If the integer is out of the 32-bit signed integer range [-2^31, 2^31 - 1], then round the integer to remain in the range. Specifically, integers less than -231 should be rounded to -2^31, and integers greater than 2^31 - 1 should be rounded to 2^31 - 1.

Return the integer as the final result.

Constraints:

  1. 0 < = s.length < = 200
  2. s consists of English letters (lower-case and upper-case), digits (0-9), ' ', '+', '-', and '.'.

難度標記：medium

解題思路

這一題有參考Leetcode解答上面解法，原因是當初寫的東西太複雜，用了太多if跟else處理

Step 1

這一題主要的解法就是把題目條件一一列出來，並依照重要性排列，最後得出一個結果

在題目的演算法中，第一個要處理的是空白，再來是正負號，接著是前面如果有0的話就要一直跳過 ，如果有讀到非0就會丟出去。最後如果超過integer的範圍，就要把答案卡在最大及最小的integer

Step 2

所以就跟著題目要求的列出code，首先要處理前面的空白，所以就設定一個i變數，代表目前掃到的字串位置

如果字串位置是空白，那就一直往前掃，直到不是空白為止

Step 3

接著處理正負號，掃完空白之後的正負號才是答案的正負號，所以就檢查如果是正的，那sign變數就是+1，如果是負的，sign變數就是-1

Step 4

處理完正負號之後就看接下來的數字是不是0~9，作法就是將ans*10+s[i](目前的值) 如果前面都是0也沒關係，因為ans = 0，且s[i]為0，得出來的答案也會是0

如果不是0~9的話，就把答案直接丟回去了，記得要把答案*sign

還要考慮到有可能掃到string的末端也沒有非0~9的digit，所以要在迴圈外面再加一個return

Step 5

至於rounding如何處理也是很簡單，只要在檢查ans的時候發現比INT_MAX或INT_MIN大或小的話，就round ans。 這也是為什麼一開始的ans要開long long int的原因。

Debug/遇到的問題

不要把例外case都列出來，有時候要想一下是否正常才是所謂的例外(有點像排列組合的反扣法)

int myAtoi(string s) {
    int i = 0;
    int sign = 1;
    long long int ans = 0;
    while(i < s.length() && s[i] == ' ')
        i++;
    if(s[i] == '-'){
        sign = -1;
        i++;
    }
    else if(s[i] == '+') i++;
    while(i<s.length()){
        if(s[i]>='0' && s[i]<='9'){
            ans = ans * 10 +(s[i]-'0');
            if(ans > INT_MAX && sign == -1) return INT_MIN;
            else if(ans > INT_MAX && sign == 1) return INT_MAX;
            i++;
        }else return ans * sign;
    }
    return (ans*sign);
}
