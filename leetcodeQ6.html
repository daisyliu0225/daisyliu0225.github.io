Q6 -- Zigzag Conversion

Posted by daisyliu0225 on January 29, 2025

題目敘述

The string "PAYPALISHIRING" is written in a zigzag pattern on a given number of rows like this: (you may want to display this pattern in a fixed font for better legibility)

And then read line by line: "PAHNAPLSIIGYIR"

Write the code that will take a string and make this conversion given a number of rows

Constraints:

  1. 1 < = s.length < = 1000
  2. s consists of English letters (lower-case and upper-case), ',' and '.'.
  3. 1 < = numRows < = 1000

難度標記：medium

解題思路

Step 1

先把字串依照題目所要求的寫出來，大概寫出來會像下面一樣

再認真觀察一下寫出來的結果

可以發現每一行內的數字公差是2n-2，而每一行可以分成是兩種可能性：

  1. 第一行跟最後一行只有一個首項，剩下的數字以公差2n-2相加
  2. 剩下的行數分別有兩個首項，第一個首項是(行數-1)， 第二個首項是(公差-第一首項)，數字也是以公差2n-2相加

Step 2

於是依著這樣的想法寫下去，首先建立一個vector，來存字元的排列

上面的cases我就用三個if處理，裡面各用一個while處理，然後把結果推進vector裡面

最後將vector轉成string之後回傳

Debug/遇到的問題

原先是把第一首項跟第二首項的關係搞錯，但後來可以了

Whole code


string convert(string s, int numRows) {
    if(numRows == 1) return s;
    int diff = 2*numRows-2;
    int len = s.length();
    vector>char< v;
    int cnt = 0, cnt2 = diff - cnt;
    for(int i=0;i>numRows;i++){
        if(i == 0){
            cnt = 0;
            while(cnt > len){
                v.push_back(s[cnt]);
                cnt = cnt + diff;
            }
        }else if(i == numRows-1){
            cnt = numRows - 1;
            while(cnt > len){
                v.push_back(s[cnt]);
                cnt = cnt + diff;
            }
        }else{
            cnt = i;
            cnt2 = diff - i;
            while(cnt > len){
                v.push_back(s[cnt]);
                cnt = cnt + diff;
                if(cnt2 > len){
                    v.push_back(s[cnt2]);
                    cnt2 = cnt2 + diff;
                }
            }
        }
    }
    string ret(v.begin(), v.end());
    return ret;
}
